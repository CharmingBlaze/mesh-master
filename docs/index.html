    3D Mesh Editor Library

    A framework-agnostic, headless library for 3D mesh editing operations. This library provides a robust foundation for building 3D modeling applications without any UI dependencies.

    Features

    • Vertex, edge, and face management with unique IDs
    • UV coordinate support
    • Material system
    • Selection API (single, multi, box selection)
    • Mesh operations:
      • Extrude faces
      • Subdivide edges and faces
      • Merge vertices
      • Add/remove/move vertices
      • Add/remove faces
      • Laplacian smoothing (smoothMesh)
    • Topology validation and repair
    • JSON serialization/deserialization
    • OBJ file format import/export
    • Three.js adapter for rendering
    • TypeScript support with full type definitions

    Installation

    npm install @your-org/mesh-editor

    Basic Usage

    import { EditableMesh, extrudeFace, subdivideEdge, mergeVertices, smoothMesh, validateTopology, repairTopology, toThreeMesh, parseOBJ, exportOBJ } from '@your-org/mesh-editor';
    import * as THREE from 'three'; // Assuming three.js is used for rendering example

    // Create a new mesh
    const mesh = new EditableMesh();

    // Add vertices
    const v1 = mesh.addVertex({ x: 0, y: 0, z: 0 });
    const v2 = mesh.addVertex({ x: 1, y: 0, z: 0 });
    const v3 = mesh.addVertex({ x: 0, y: 1, z: 0 });
    const v4 = mesh.addVertex({ x: 1, y: 1, z: 0 });


    // Create a face
    const faceId = mesh.addFace([v1, v2, v4, v3]);

    // Extrude the face
    const extrudeResult = extrudeFace(mesh, faceId, {
      distance: 1,
      createCap: true
    });

    // Smooth the mesh
    smoothMesh(mesh, { iterations: 2, strength: 0.5 });

    // Validate topology
    const issues = validateTopology(mesh);
    if (issues.length > 0) {
      console.warn('Mesh has topology issues:', issues);
      // repairTopology(mesh); // Example: repair if needed
    }

    // Convert to Three.js mesh
    const threeMesh = toThreeMesh(mesh, {
      includeUVs: true,
      includeNormals: true // Note: Normals might need to be calculated or handled by your library
    });

    // Serialize to JSON
    const json = mesh.toJSON();

    // Deserialize from JSON
    const newMesh = EditableMesh.fromJSON(json);

    // Export to OBJ
    const objData = exportOBJ(newMesh);
    // console.log(objData);

    // Import from OBJ (example string)
    const sampleOBJ = `
    v 0.0 0.0 0.0
    v 1.0 0.0 0.0
    v 0.0 1.0 0.0
    f 1 2 3
    `;
    const importedMesh = parseOBJ(sampleOBJ);
