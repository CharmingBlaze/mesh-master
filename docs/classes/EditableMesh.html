    Class EditableMesh

    Core mesh editing class that manages vertices, edges, faces, UVs, and materials. This class provides a robust foundation for 3D mesh manipulation with built-in validation and error handling.

    Example

    const mesh = new EditableMesh();
    const v1 = mesh.addVertex({ x: 0, y: 0, z: 0 });
    const v2 = mesh.addVertex({ x: 1, y: 0, z: 0 });
    const v3 = mesh.addVertex({ x: 0, y: 1, z: 0 });
    const face = mesh.addFace([v1, v2, v3]);

    • addFace(vertexIds: number[], materialId?: null | number): number

      Adds a new face with the given vertices. Creates necessary edges and validates the input data.

      Parameters

      • vertexIds: number[]

        Array of vertex IDs that form the face

      • materialId: null | number = null

        Optional material ID to assign to the face

      Returns number

      The ID of the newly created face

      Throws

      Error if vertices are invalid or face cannot be created

      // Create a triangular face
